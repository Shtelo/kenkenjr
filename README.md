# kenkenjr

kenkenjr은 슈텔로의 관리를 위해 새로운 봇을 개발하는 프로젝트이다.

## 기록

기록의 각 행은 특수한 의미를 갖는 문자로 시작하며 한 문장으로 간결히 서술한다.

* `+`: 기능 추가
* `-`: 기능 제거
* `~`: 기능 수정
* `!`: 오류 발견
* `@`: 오류 수정
* `#`: 주석
* `\t`: 세부사항
